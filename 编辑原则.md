Minecraft联机平台开发规范

1.HTMl中绝对不能有任何的js代码，所有js代码只能被引用
2.各部分的js由各自的js文件处理
3.依据supabaseinfo文件编写数据名

文件结构
public/
├── index.html # 主发布页面
├── post.html # 帖子详情页
├── browse.html # 帖子浏览页
├── admin.html # 管理后台
js/
├── supabase.js # 数据库连接配置
├── post.js # 发帖功能核心逻辑
├── list.js # 帖子列表渲染
├── interaction.js # 用户交互处理
├── filter.js # 筛选功能实现
├── admin.js # 管理功能模块
├── utils.js # 通用工具函数
style.css # 全局样式表
